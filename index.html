Negar

Negar trying to be a spell corrector for persian language.

Welcome to Negar Pages.

Negar will not do any special thing except try to help you in write a more beautiful Persian text. for now it helps you in editing your pages with legal space character & complete Persian alphabet. In the future it will have a spell corrector engine and a graphical user interface (GUI).

how to use it:

$ git clone https://github.com/shahinism/Negar.git
$ cp Negar/Virastar.py /usr/bin
$ Virastar.py FILENAME > output

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
